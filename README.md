# electron-react-typescript

![](./aplicacao.png)

## Intalação

Utilize o yarn para instalar as dependências

```bash
yarn
```
Instale o banco de dados Mongodb 

```bash
yarn add mongodb    
```

## Esxecução

Para a execução do programa basta executar o comando.

```bash
yarn dev
```
